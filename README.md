# Park&Go - Smart Parking App Concept 🚗📱

**Park&Go** is a conceptual mobile application and business model developed in 2021 as a high school group project (Liceo Scientifico, Cuneo). The project tackles urban mobility challenges by designing a remote system for booking and paying for parking spaces.

## 💡 The Concept
Finding parking in busy urban areas is a common source of stress and wasted time. *Park&Go* was conceptualized to solve this by offering:
* Real-time location of available parking spots.
* Remote booking and digital payment integration.
* Subscription management for frequent commuters.
* A user-friendly, intuitive mobile interface.

## 🎨 UI/UX Prototype Showcases
Although this project did not involve backend programming, a complete high-fidelity mockup of the user interface was designed to simulate the user journey.

**Key Screens Designed:**
* **Authentication:** Login and Registration flows.
* **Navigation:** Main interactive menu and integrated Search Map.
* **Management:** Active reservations, subscription plans, and user profile settings.
* **Support:** FAQ and "About Us" informational sections.
